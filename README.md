# The Hollow Script

A beautifully crafted digital journal where silence bleeds ink and emotion finds voice.

## Overview

The Hollow Script is an elegant, minimalist web-based journal experience. It presents poetic content in an intimate digital space with carefully designed typography and a sophisticated color palette.

## Features

- **Minimalist Design**: Clean, distraction-free reading experience
- **Custom Typography**: Elegant font choices including Cormorant Garamond and Special Elite
- **Responsive Layout**: Optimized for multiple screen sizes
- **Organized Content**: Structured journal entries with chapter navigation
- **Atmospheric Aesthetics**: Carefully curated color scheme and visual hierarchy

## Project Structure

```
├── index.html          # Landing page with journal cover
├── contents.html       # Table of contents with chapter listings
├── false9.html         # Chapter content page
├── illusion.html       # Chapter content page
├── thechoir.html       # Chapter content page
├── style.css           # Global styles and theming
└── README.md           # This file
```

## Files Description

- **index.html**: The main landing page featuring the journal's title and invitation to open the journal
- **contents.html**: A curated index of all available entries/chapters
- **HTML Content Pages**: Individual pages for different entries (false9.html, illusion.html, thechoir.html)
- **style.css**: Master stylesheet containing all design rules, color variables, and responsive breakpoints

## Color Scheme

The project uses a curated palette of literary colors defined in the CSS:

- Primary ink colors for text
- Faded ink for secondary text
- Blood-dry red accents for borders and highlights

## Getting Started

1. Open `index.html` in a web browser to start reading
2. Click "open the journal" to navigate to the contents page
3. Select chapters from the contents to read individual entries

## Browser Compatibility

Designed to work across all modern browsers supporting HTML5 and CSS3.

## Design Philosophy

The Hollow Script embraces minimalism and literary aesthetics, creating a space where written emotion can be experienced without digital noise or distraction.
